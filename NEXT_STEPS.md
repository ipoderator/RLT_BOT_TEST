# Следующие шаги

## Текущий статус

✅ **Python 3.11.7** - версия подходит  
✅ **Все зависимости установлены:**
- aiogram
- sqlalchemy
- asyncpg
- psycopg2
- openai

⚠️ **Требуется настройка:**
- Файл `.env` не найден

---

## Шаг 1: Создание файла .env

### Вариант 1: Интерактивная настройка (рекомендуется)

```bash
python setup_env.py
```

Скрипт поможет заполнить все необходимые переменные окружения.

### Вариант 2: Ручная настройка

```bash
cp .env.example .env
```

Затем отредактируйте `.env` файл и заполните значения:
- `TELEGRAM_BOT_TOKEN` - токен бота от @BotFather
- `OPENAI_API_KEY` - API ключ OpenAI
- `DATABASE_URL` - URL базы данных PostgreSQL

---

## Шаг 2: Настройка PostgreSQL

1. Убедитесь, что PostgreSQL установлен и запущен
2. Создайте базу данных:
   ```sql
   CREATE DATABASE video_analytics;
   ```
3. Проверьте подключение (обновите DATABASE_URL в .env при необходимости)

---

## Шаг 3: Загрузка данных

1. Скачайте JSON файл с данными (ссылка должна быть предоставлена отдельно)
2. Загрузите данные в базу:
   ```bash
   python load_data.py <путь_к_json_файлу>
   ```

---

## Шаг 4: Проверка структуры БД

```bash
python test_database_structure.py
```

Должна быть успешно пройдена проверка всех таблиц и колонок.

---

## Шаг 5: Запуск проверок

После настройки всех переменных окружения запустите полную проверку:

```bash
python run_all_tests.py
```

Этот скрипт выполнит все проверки требований и выведет итоговый отчет.

---

## Шаг 6: Запуск бота

После успешного прохождения проверок запустите бота:

```bash
python bot.py
```

---

## Быстрая проверка готовности

Выполните команду для проверки всех предварительных условий:

```bash
python run_all_tests.py
```

Если все переменные окружения настроены, скрипт запустит все проверки автоматически.

---

## Полезные команды

```bash
# Проверка структуры БД
python test_database_structure.py

# Проверка генерации SQL
python test_sql_generation.py

# Тестирование примеров вопросов
python test_bot.py

# Комплексная проверка всех требований
python test_requirements.py

# Запуск всех проверок
python run_all_tests.py
```
